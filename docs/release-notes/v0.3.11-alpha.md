# v0.3.11-alpha

Study Anything v0.3.11-alpha adds a marketplace-quality external Agent eval
harness for Kimi-compatible, Codex Skill, WorkBuddy-style HTTP, and generic
OpenAPI platform submissions.

## What's New

- Adds `external-eval-marketplace-harness-v1`, a redacted eval contract for
  platform operators and marketplace-style reviewers.
- Separates required native release gates from optional mature eval adapters:
  Promptfoo, DeepEval, LangChain AgentEvals, and Ragas-compatible retrieval.
- Includes fake and mock HTTP Agent fixtures, sample eval cases, timeout policy,
  expected evidence schema, failure remediation, and privacy assertions.
- Wires the harness into platform packs, ecosystem submission metadata,
  adoption packs, bundle manifests, operator drill evidence, manual-submission
  rehearsal, release checks, and external adoption proof.

## Known Limits

- Promptfoo, real DeepEval, full Ragas, and LangChain AgentEvals runtime
  packages remain optional unless an operator explicitly requires them.
- The harness proves adapter readiness, redaction, and release-gate wiring; it
  is not an official approval from any external platform.
- Real model and judge credentials remain in the user's own Agent or eval
  environment, never inside Study Anything.
