# v0.2.10-alpha

Study Anything v0.2.10-alpha adds the Agent Eval foundation needed for platform-agent and Skill-based
distribution. The release keeps the API/Skill launch path from v0.2.9-alpha and adds a redacted eval
artifact bridge for mature open-source eval tools.

## Highlights

- Adds `GET /v1/sessions/{session_id}/agent-eval/artifact`, derived from Agent audit evidence.
- Adds native no-judge gates for Agent invocation coverage, source binding, excerpt hashing, and
  privacy redaction.
- Adds adapter strategy metadata for Promptfoo, DeepEval, LangChain AgentEvals, and Ragas.
- Adds a Promptfoo contract-gate template at `evals/promptfoo/agent-eval-artifact.yaml`.
- Adds `scripts/verify_agent_eval_flow.py` for fake-agent and user-owned HTTP Agent smoke validation.
- Extends full API and mock HTTP Agent smoke scripts to verify Agent Eval artifact readiness.
- Updates default published-image tags to `v0.2.10-alpha`.

## First Launch

Install Docker Desktop or Docker Engine with the Compose plugin, start the daemon, then run:

```bash
python3 scripts/setup_env.py
USE_PUBLISHED_IMAGES=true ./scripts/launch_self_host.sh
```

Open:

- API docs: http://localhost:8000/docs
- API health: http://localhost:8000/v1/health
- System status: http://localhost:8000/v1/system/status

Maintainers can verify the public API image end to end with:

```bash
python3 scripts/verify_published_image_launch.py --tag v0.2.10-alpha
```

## Agent Eval Smoke

Against a running API:

```bash
API_BASE=http://127.0.0.1:8000 python3 scripts/verify_agent_eval_flow.py
```

For a user-owned HTTP Agent path, start an Agent endpoint first:

```bash
python3 scripts/mock_http_agent.py --host 127.0.0.1 --port 8787
EXPECT_EXTERNAL_AGENT=true API_BASE=http://127.0.0.1:8000 AGENT_ENDPOINT=http://127.0.0.1:8787/invoke \
  python3 scripts/verify_agent_eval_flow.py
```

## Upgrade

Create a local backup before upgrading:

```bash
python3 scripts/self_host_data.py backup
```

Then pull the release image:

```bash
STUDY_ANYTHING_IMAGE_TAG=v0.2.10-alpha \
USE_PUBLISHED_IMAGES=true \
./scripts/launch_self_host.sh
```

## Known Limitations

- The alpha remains self-host-first. Managed hosting, accounts, billing, Publish, and Teams are not
  included yet.
- There is no shippable standalone Web UI in this release. Future UI work belongs on a separate branch
  after the API/Skill path is stable.
- Agent Eval artifact proves contract readiness and invocation coverage. DeepEval, LangChain
  AgentEvals, and Ragas quality suites still need dedicated eval datasets and judge-model operator
  configuration.
- Study Sync is still a local encrypted package boundary, not hosted remote sync.
- Recovery remains manual. API status surfaces do not execute destructive restore operations.
- Plugin installation remains local and explicit; remote marketplace distribution is not implemented.
- First-run GHCR downloads can take several minutes on slower connections.
