# v0.3.2-alpha

Study Anything v0.3.2-alpha adds the local adoption telemetry and PMF readiness layer for
platform-Agent distribution.

## Added

- `adoption-telemetry-v1` via `GET /v1/adoption/telemetry`.
- `pmf-readiness-v1` via `GET /v1/pmf/readiness`.
- `scripts/verify_adoption_telemetry.py`, including fixture checks that reject source text, answers,
  insights, raw user ids, Agent endpoints, API keys, and browser/video/app private context.
- Platform tools for adoption telemetry and PMF readiness in the OpenAPI/OpenAI tool manifests.
- Adoption pack, ecosystem submission, release checklist, and docs coverage for the telemetry
  contract.

## Changed

- `pmf-export-v1` now embeds `adoption-telemetry-v1` and `pmf-readiness-v1` after explicit consent.
- External adoption proof runs the telemetry verifier as part of the platform evidence chain.
- Release defaults now point to `v0.3.2-alpha`.

## Boundaries

- No automatic upload is added.
- No standalone frontend is required.
- Hosted Sync, Publish, Teams, Catalyst, billing, remote accounts, and marketplace payments remain
  future work until PMF evidence is stronger.
