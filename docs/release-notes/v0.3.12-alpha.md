# v0.3.12-alpha

Study Anything v0.3.12-alpha turns the plugin ecosystem into a copy-ready
platform adoption kit. The goal is not a hosted marketplace yet; it is a
local-first trust path that Kimi, Codex, WorkBuddy, or another platform Agent
can inspect before suggesting any plugin install.

## Added

- Adds `plugin-ecosystem-adoption-kit-v1`, a redacted report that verifies:
  - bundled sample plugin manifests and source files;
  - `plugins/registry.json` sourceDigest alignment;
  - quarantine-first install policy;
  - platform pack verification commands and acceptance evidence;
  - no plugin entrypoint execution during validation.
- Includes the bundled sample plugin source and registry in the platform
  adoption pack:
  - `example-note-importer`
  - `example-web-importer`
  - `example-enrichment-importer`
  - `example-exporter`
  - `example-agent-provider`
- Wires the plugin ecosystem verifier into ecosystem submission, platform packs,
  bundle manifests, manual rehearsal, operator drill, external adoption proof,
  and release checks.

## Trust Boundary

- Study Anything still does not download remote plugin code, store real model
  keys, or execute plugin entrypoints during preview, validation, quarantine, or
  registry review.
- Community plugin installation remains an explicit local-directory operation.
- Hosted plugin review, payments, automatic updates, and marketplace
  distribution remain future commercial trust-layer work.
