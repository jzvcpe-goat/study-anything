# v0.2.29-alpha

Study Anything v0.2.29-alpha hardens Agent Eval maturity for platform-agent and user-owned Agent
workflows. The release keeps the product API/Skill Mode first: no standalone frontend is required,
and real model credentials stay in the user's Agent or eval environment.

## Added

- `GET /v1/evals/policy` returns `agent-eval-policy-v1`: native release gates, optional external
  adapter policy, failure classes, fixture paths, and privacy rules.
- `GET /v1/sessions/{session_id}/agent-eval/report` returns `agent-eval-report-v1`: invocation,
  trajectory, teaching quality, retrieval grounding status, export readiness, privacy, and adapter
  readiness for one completed learning loop.
- `scripts/run_external_agent_evals.py --tool report` provides a fast native maturity gate that does
  not require Promptfoo, DeepEval, Ragas, judge models, or provider keys.
- `evals/fixtures/fake-agent-learning-loop.json` and
  `evals/fixtures/mock-http-agent-learning-loop.json` provide redacted fixtures for fake and
  user-owned HTTP Agent loops.

## Changed

- Platform tools now include `study_anything_eval_policy` and `study_anything_agent_eval_report`.
- Kimi, Codex, WorkBuddy packs, Skill Mode docs, and the adoption pack include the new eval policy
  and report gates.
- The deterministic Agent eval baseline now includes `agent-eval-report-v1`.
- Release defaults now point to `v0.2.29-alpha`.

## Privacy And Security

- Study Anything still never stores real model API keys, judge-model keys, Agent endpoints, or
  external eval credentials.
- Eval policy, artifact, quality, report, fixtures, and baseline remain redacted: no raw source text,
  learner answers, grading feedback, generated insights, Agent endpoints, raw Agent metadata, or
  secrets.
- External Promptfoo, DeepEval, LangChain AgentEvals, and Ragas runs are optional unless a release
  operator explicitly marks them required.

## Known Limits

- `agent-eval-report-v1` is a deterministic maturity gate, not a full pedagogy benchmark.
- Promptfoo, DeepEval, LangChain AgentEvals, and Ragas adapters remain optional external gates owned
  by the operator's local or CI environment.
- Hosted Sync, Teams, Publish, billing, marketplace payments, and a standalone frontend remain future
  commercial surfaces.
