# Study Anything v0.3.21-alpha

This alpha packages the v0.3.20 public support status into a single external
adopter evidence archive and maintainer handoff bundle.

## Added

- `adopter-evidence-archive-v1` generated JSON, Markdown, zip, and checksum.
- `adopter-evidence-fixture-v1` fixtures for successful release, local GHCR
  pull timeout, needs-repro, release-blocker, platform-blocked, and resolved
  support states.
- `docs/adopter-evidence-archive.md`.
- Release-gate, adoption-pack, ecosystem-submission, operator-drill, manual
  rehearsal, platform-pack, and public status wiring for the archive.

## Verification

```bash
python3 scripts/generate_adopter_evidence_archive.py --check
python3 scripts/verify_adopter_evidence_archive.py --check
python3 scripts/verify_adopter_evidence_archive.py \
  --pack platform/generated/study-anything-platform-adoption-pack.zip
./scripts/release_check.sh
```

## Boundaries

- The archive is public metadata only.
- No standalone frontend is required.
- No real model keys are stored by Study Anything.
- No raw source text, learner answers, Agent prompts, Agent endpoints,
  browser/video/app private context, support bundle private payloads, or
  personal profile data are included in adopter evidence.
- Hosted Sync, Publish, Teams, billing, SSO, and marketplace payments remain
  post-PMF work.
