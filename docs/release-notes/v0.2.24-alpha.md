# v0.2.24-alpha

Study Anything v0.2.24-alpha is the Agent Eval maturity release. It turns the
existing redacted audit/eval artifacts into a committed, repeatable regression
gate that can run locally and in CI without judge-model keys.

## Highlights

- Added `scripts/verify_agent_eval_baseline.py`, a deterministic native
  baseline and comparison gate for Agent eval quality.
- Added `evals/baselines/study-anything-agent-eval-baseline.json` with a
  redacted scorecard covering adapter ids, trajectory coverage, native gates,
  teaching quality, retrieval/context quality, and privacy invariants.
- Wired the baseline gate into CI, `scripts/release_check.sh`, external
  adoption proof, platform bundle/adoption pack generation, and eval asset
  verification.
- Kept Promptfoo, DeepEval, LangChain AgentEvals, and Ragas as mature ecosystem
  targets while preserving local-first behavior: external installs are optional
  unless the operator explicitly marks them required.
- Kept real model and judge-model credentials outside Study Anything.

## Validation Target

Before tagging this release:

```bash
python3 scripts/verify_agent_eval_baseline.py --check
python3 scripts/verify_agent_eval_assets.py
python3 scripts/verify_external_adoption.py \
  --pack platform/generated/study-anything-platform-adoption-pack.zip \
  --copy-worktree
./scripts/release_check.sh
python3 scripts/verify_published_image_launch.py --tag v0.2.24-alpha
```

## Known Limits

- The baseline is deterministic and contract-focused. It is not a substitute
  for a real judge-model pedagogy benchmark.
- Promptfoo, real DeepEval, full Ragas, and LangChain AgentEvals package runs
  remain optional external gates with explicit timeouts.
- Study Anything still does not store real model, platform, or judge-model API
  keys.
