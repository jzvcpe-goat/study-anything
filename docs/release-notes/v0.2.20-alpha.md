# v0.2.20-alpha

Study Anything v0.2.20-alpha turns importer plugins into a controlled runtime and adds an optional
retrieval projection for platform-agent learning flows.

## Added

- Added `POST /v1/importers/{plugin_id}/run` for reviewed local importer plugins.
- Added CLI `importer-run` with permission confirmation, optional package output, and optional
  session creation or expansion.
- Added optional retrieval APIs:
  - `GET /v1/retrieval/status`
  - `POST /v1/sessions/{session_id}/retrieval/rebuild`
  - `GET /v1/sessions/{session_id}/retrieval/search`
  - `POST /v1/sessions/{session_id}/retrieval/search`
  - `POST /v1/sessions/from-retrieval`
  - `POST /v1/sessions/{session_id}/retrieval/context-package`
- Added CLI retrieval commands: `retrieval-status`, `retrieval-rebuild`, `retrieval-search`, and
  `retrieval-import`.
- Added optional LanceDB dependency and a local smoke-only memory retrieval backend.
- Added `scripts/verify_importer_runtime_retrieval_flow.py` to prove importer runtime -> retrieval ->
  lesson -> quality eval -> Obsidian/learning-package export.
- Expanded platform tool assets for Codex, Kimi, and WorkBuddy-style Agents with importer runtime and
  retrieval tools.

## Security And Privacy

- Importer discovery, preview, install, and registry review remain metadata-only.
- Importer execution requires exact `confirmed_permissions`.
- Importers requesting `network:http` are blocked unless the caller sends `allow_network=true`.
- Importer output must validate as `learning-context-package-v1` before it can enter learning state.
- Retrieval is rebuildable from canonical session state and stores only references, excerpt hashes,
  locators, minimal snippets, vectors, and timestamps.
- Study Anything still does not store real model API keys, Agent secrets, Agent endpoints inside
  exports, or browser/application credentials.

## Known Limitations

- The importer runtime is a narrow Python entrypoint loader, not a general sandbox.
- The memory retrieval backend is for smoke tests only and is not durable.
- Default deterministic embeddings are local hash vectors for adoption proof; external Agent
  embeddings and retrieval quality evals are planned next.
- Live NotebookLM sync is still not implemented; NotebookLM support remains a package/export bridge.
