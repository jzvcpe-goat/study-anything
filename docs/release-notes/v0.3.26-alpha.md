# Study Anything v0.3.26-alpha

This alpha adds a platform-agent release replay simulator. The goal is to let a
Kimi, Codex, WorkBuddy, or generic OpenAPI operator start from GitHub Release
assets, import the Study Anything tool contract, and prove the minimum learning
tool chain can be called against a running API.

## Highlights

- Added `scripts/replay_platform_agent_from_release.py`, which unpacks release
  assets, validates platform entrypoints, imports OpenAI/OpenAPI tool contracts,
  and replays the minimum learning workflow.
- Added `platform-agent-release-replay-v1` evidence assets and a top-level
  `study-anything-platform-agent-replay.zip` release asset.
- Covered Kimi, Codex, WorkBuddy, and generic OpenAPI platform profiles with
  metadata-only, Skill Mode, external API, and published-image replay modes.
- Kept replay transcripts redacted: no raw source text, learner answers, Agent
  prompts, Agent endpoint secrets, model keys, support payloads, or local paths.

## Validation

```bash
python3 scripts/replay_platform_agent_from_release.py --tag v0.3.26-alpha --platform kimi --runtime metadata-only
python3 scripts/replay_platform_agent_from_release.py --fixture fixtures/release-asset-adoption/asset-only-pass.json --asset-dir platform/generated --platform kimi --runtime metadata-only
python3 scripts/generate_platform_agent_replay.py --check
python3 scripts/bootstrap_from_release.py --tag v0.3.26-alpha --runtime metadata-only
python3 scripts/verify_release_asset_adoption.py --tag v0.3.26-alpha --runtime metadata-only
python3 scripts/generate_platform_adoption_pack.py --check
python3 scripts/verify_ecosystem_submission_pack.py
./scripts/release_check.sh
```

## Known Limits

- Metadata-only replay proves release asset integrity and tool import readiness.
  It does not call a runtime unless `--runtime skill-mode`, `--runtime
  external-api`, or `--runtime published-image` is selected.
- Real model choice remains outside Study Anything. The user's platform Agent or
  local Agent gateway owns model credentials, browser access, and external tools.
