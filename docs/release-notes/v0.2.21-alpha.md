# v0.2.21-alpha

Study Anything v0.2.21-alpha turns the platform-agent integration work into a stronger ecosystem
quality gate. The release keeps Study Anything local-first and API/Skill-first while adding redacted
retrieval/context eval evidence for Kimi, Codex, WorkBuddy-style workspaces, and future Ragas-style
quality suites.

## Highlights

- Added `retrieval-quality-eval-v1` for retrieval availability, result count, source binding, snippet
  minimality, query relevance, context-package validity, and privacy invariants.
- Added `GET /v1/evals/retrieval/cases` and `GET|POST /v1/sessions/{session_id}/retrieval/eval`.
- Added CLI support with `python3 scripts/study_anything_cli.py retrieval-eval SESSION_ID --query ...`.
- Extended `scripts/run_external_agent_evals.py` with `--tool retrieval`, a Ragas-compatible native
  retrieval quality adapter.
- Added `scripts/verify_platform_ecosystem_eval_flow.py` to prove platform Agent context collection,
  importer runtime, enrichment, retrieval, teaching layers, learning loop, external eval adapters,
  Obsidian export, and learning-package export in one flow.
- Updated platform tools, generated OpenAPI/OpenAI assets, and Kimi/Codex/WorkBuddy packs with the
  new retrieval quality gate.

## Privacy Boundary

- Retrieval eval reports do not include raw source text, learner answers, grading feedback, Agent
  endpoints, raw Agent metadata, secrets, or retrieval snippets.
- Promptfoo, DeepEval, and Ragas-style judge credentials remain outside Study Anything in the user's
  platform Agent or eval runner environment.
- LanceDB remains optional and rebuildable; canonical learning state remains in the Study Anything
  session store.

## Known Limitations

- The retrieval quality gate is deterministic. It proves contract, privacy, and minimum context
  quality, not full semantic pedagogy.
- Ragas integration is adapter-ready through the redacted report, but not a default installed
  dependency or required CI path.
- Browser-only Kimi still needs a terminal-capable Agent, local gateway, or private workspace tool
  runner to call localhost.
