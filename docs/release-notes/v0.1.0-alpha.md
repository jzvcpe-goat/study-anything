# v0.1.0-alpha

Initial alpha target.

## Included

- OSS project foundation.
- Deterministic learning workflow.
- Agent registry and fake demo agent.
- FastAPI routes.
- React/Vite UI shell.
- Docker Compose self-host stack.
- Plugin manifest validation and example plugin.
- Postgres-backed Docker session store.
- GitHub Actions for API tests, Web build/audit, Compose smoke, and GHCR image publishing.
- Local `.env` secret generation and validation scripts.

## Known Limitations

- LangGraph and Postgres checkpointer dependencies are declared but the alpha executor is still a local deterministic implementation.
- Agent provider defaults are JSON-backed in the local Docker volume during alpha.
- Langfuse container stack is present; node trace emission requires project keys and is staged for v0.2.
- LanceDB, Mem0, LangMem, and FSRS are roadmap integrations, not v0.1 launch blockers.
- Hosted Sync, Publish, Teams, and Catalyst are roadmap items.
